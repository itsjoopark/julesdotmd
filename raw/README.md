# Raw Sources

Immutable source documents. The LLM reads from this directory but never modifies it. You add files here; the LLM processes them into the wiki.

- `articles/` — web clippings (Obsidian Web Clipper output, saved articles)
- `notes/` — journals, meeting notes, podcast notes, personal writing
- `papers/` — PDFs, academic papers, reports
- `assets/` — downloaded images referenced by sources
